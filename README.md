# android-support-business
* Android Module SDK(业务模块)
- 建议使用像B站，组织的形式创建仓库
- [AndroidKnife](https://github.com/AndroidKnife)

## 特性(feature)
* 基于Android API and Google API
* 不依赖第三方类库
* 组件，功能导向

## 效用(utility)
* 基类，抽象，接口
* 可复用(Reuseable)，快速开发，不涉及具体的业务
